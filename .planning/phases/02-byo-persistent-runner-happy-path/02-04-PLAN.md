<interfaces>
No new public code interface is required. This plan hardens the final user-facing contract:

- Command: `runnerkit up --repo owner/name --host user@host`
- Supported happy path: trusted private repository, BYO Linux systemd host, persistent runner, non-root service user `runnerkit-runner`.
- Recommended workflow snippet shape: `runs-on: [self-hosted, runnerkit, runnerkit-owner-repo, linux, x64, persistent]`.
- Out of scope: cloud provisioning, cleanup/recovery/doctor, ephemeral mode, automatic workflow YAML edits.
  </interfaces>
