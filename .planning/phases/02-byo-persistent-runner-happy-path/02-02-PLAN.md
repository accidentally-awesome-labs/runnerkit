<objective>
Implement the BYO Linux bootstrap installer that prepares dependencies, creates a non-root runner account, downloads and verifies the official GitHub Actions runner, configures it, installs a systemd service, and verifies service readiness through the remote executor seam.

Purpose: Phase 2's happy path depends on RunnerKit performing the manual host setup safely and repeatably after Plan 02-01 preflight passes.

Output: A tested `internal/bootstrap` package and CLI integration that can render/apply the remote installer with fake remote execution, preserving non-root service and token-redaction guarantees.
</objective>
