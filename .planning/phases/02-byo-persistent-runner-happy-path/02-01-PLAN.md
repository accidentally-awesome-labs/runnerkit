<objective>
Add the Phase 2 BYO SSH intake and remote preflight foundation for `runnerkit up`.

Purpose: RunnerKit cannot safely install or register a persistent runner until it can identify the SSH target, verify host identity, persist accepted fingerprints, and prove the remote Linux/systemd host is compatible enough to mutate.

Output: A testable `internal/remote` boundary, `internal/preflight` checks, state fields for host trust, and `runnerkit up` wiring that stops before mutation when preflight is missing, failed, or untrusted.
</objective>
